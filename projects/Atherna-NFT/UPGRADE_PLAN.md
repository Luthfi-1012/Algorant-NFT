Karena spesifikasi memerlukan Next.js 14, kita punya 2 opsi:

1. **Migrate ke Next.js 14** (Recommended untuk production)
   - Setup Next.js 14 project baru
   - Copy semua komponen dan logic
   - Implement semua fitur baru

2. **Upgrade Vite project** dengan semua fitur
   - Tetap menggunakan Vite (lebih cepat untuk development)
   - Install semua dependencies yang diperlukan
   - Implement semua fitur sesuai spesifikasi

Karena proyek sudah berjalan dengan Vite dan struktur cukup baik, kita akan upgrade Vite project dengan semua fitur yang diminta, sambil tetap kompatibel dengan struktur yang ada.
